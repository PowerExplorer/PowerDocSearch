# PowerDocSearch
1) Search all text, document files (pdf, odt, doc, docx, rtf, epub, fb2, chm, ppt, pptx, odp, ods, xls, xlsx...) 
2) Search documents in compressed files (zip, 7z, rar, xz, cab, bz2, tar, gz...)
